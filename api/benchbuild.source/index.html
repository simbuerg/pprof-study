</>

Provide a base interface for downloadable sources.

Classes
  • Variant — Provide a 'string'-like wrapper around source version information.</>
  • Fetchable — Base class for protocol classes.</>
  • Expandable — Base class for protocol classes.</>
  • Versioned — Base class for protocol classes.</>
  • FetchableSource(local, remote) — Base class for fetchable sources.</>
  • NoSource — Base class for fetchable sources.</>
Functions
  • context(*variants) (dict(str: Variant)) — Convert an arbitrary number of variants into a VariantContext.</>
  • default(*sources) (dict(str: Variant)) — Return the collective 'default' version for the given sources.</>
  • primary(*sources) (SourceT) — Return the implicit 'main' source of a project.</>
  • product(*sources) (iterable of (any, ...)) — Return the cross product of the given sources.</>
  • sources_as_dict(*sources) (dict(str: Fetchable)) — Convert fetchables to a dictionary.</>
  • target_prefix() (str) — Return the prefix directory for all downloads.</>
  • to_str(*variants) (str) — Convert an arbitrary number of variants into their string representation.</>
